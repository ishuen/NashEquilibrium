Nash Equilibrium for Simultaneous Move and Complete Information
=====

### Input

(a) user input

First, type the command "\<INPUT\>", and then input the matrix start from the next line.         
For example:
> (2,3), (-1,5)
> 
> (3,2), (1,1)

(b) file input

 \<READ\> fileName.txt

### This program only considers the square matrix.
